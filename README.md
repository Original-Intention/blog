### 如何使用
```
$ git clone https://github.com/withstars/Blog-System

$ cd Blog-System

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run

http://localhost:8080
http://localhost:8080/admin 
```
